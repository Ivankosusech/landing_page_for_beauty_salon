# 💄 L'Esthétique — Landing Page для салона красоты

[🌐 Посмотреть демо](https://ivankosusech.github.io/landing_page_for_beauty_salon/)

> Премиальный лендинг для салона красоты и стиля «L'Esthétique». Современный дизайн, адаптивная вёрстка и удобная форма записи онлайн.

---

## 📋 О проекте

Одностраничный сайт (landing page) для салона красоты, включающий:

- 🎨 Элегантный дизайн в премиум-стиле с французским акцентом
- 📱 Полностью адаптивная вёрстка (mobile-first)
- ✨ Плавные анимации и интерактивные элементы
- 📝 Форма записи на услуги с валидацией
- 🗂️ Структурированные разделы:
  - Главный экран с призывом к действию
  - О салоне и философии бренда
  - Каталог услуг с ценами
  - Команда мастеров с фото и описанием
  - Галерея работ
  - Контакты и карта
  - Футер с соцсетями и юридической информацией
